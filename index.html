React Native Beginner's Guide

Learn how to create your first React Native app from scratch

Step-by-Step Guide

Prerequisites

  • Node.js installed (version 12 or newer)
  • JavaScript and React basic knowledge
  • Java Development Kit (JDK) for Android development
  • Xcode for iOS development (macOS only)

1Install React Native CLI

Open your terminal and run the following command to install the React Native command-line interface globally:

npm install -g react-native-cli

2Create a New Project

Navigate to your development directory and create a new React Native project:

react-native init MyFirstApp

Replace "MyFirstApp" with your desired project name.

3Navigate to Project Directory

Change into your project directory:

cd MyFirstApp

4Run the Application

Start the development server and run your app on an emulator or physical device:

react-native run-android # For Android
react-native run-ios # For iOS (macOS only)

5Understand the Project Structure

Key files and directories in your React Native project:

App.js # Main component file
index.js # Entry point
android/ # Android native code
ios/ # iOS native code
package.json # Project dependencies

6Modify Your App

Edit the App.js file to create your custom UI. Here's a simple example:

import React from 'react';
import { View, Text, StyleSheet, Button } from 'react-native';

const App = () => {
  return (
    <View style={styles.container}>
      <Text style={styles.title}>Welcome to React Native!</Text>
      <Button title="Get Started" onPress={() => alert('Hello World!')} />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#6a11cb'
  },
  title: {
    fontSize: 24,
    color: 'white',
    marginBottom: 20
  }
});

export default App;

7Test Your Changes

After saving your changes, reload the app in your emulator/device:

  • Android: Press R twice or run adb shell input keyevent 82
  • iOS: Press Cmd+R in the iOS simulator
